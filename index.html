Instructions

Note: This demo works in the latest versions of Chrome, Firefox, and Internet Explorer.
  • Select a node by clicking it.
  • Edit a node's contents by clicking inside it after it is selected.
    • Rich HTML content is supported via keyboard shortcuts (Ctrl+B for bold, etc) or by pasting text or images from the clipboard.
  • Add a child node by clicking the plus button () on a selected node.
  • Delete a node by dragging it to the trash bin ().
  • Change the parent of a node by dragging it onto another node.
  • Make a new root node by dragging it into an empty area.
  • Reorder nodes by dropping them on top of its parent to move them to the right of their siblings.
